Raspberry Pi 4

Der Pi 4 ist die neuste Iteration des kleinen single-board Computers von der Raspberry Pi Foundation. Der Pi 4 verfügt über viele verschiedene Anschlüsse, und könnte hypotetisch neben dem kleinen Video-Anschluss, den wir für das Display verwenden werden, noch zwei externe Monitore bis 4k ansteuern.
Die Kabel lassen sich bequem aus dem Tablet führen, wenn Sie ein kleines Loch in eine der Seiten stechen.

Pro
+2X Micro HDMI +USB-C stromanschluss
+Ethernet-Anschluss +Programmierbare GPIO-Pins
+2X USB 3 Ports +2X USB 2 Ports
+Wlan und Bluetooth
Contra
—Relativ hoher Preis —Aktueller (2022) Preisanstieg

Wählen Sie diesen Computer, wenn Sie ein solides, und zuverlässiges Produkt wollen.
(Ich empfehle eine Version mit mindestens 2GB Ram)
Zur Website

Raspberry Pi 3

Der Pi 3 ist der Vorgänger des Pi 4 , ist jedoch weitgehend mit dem Raspberry Pi 4 kompatibel.

Pro
+HDMI-Anschluss +Etwas billiger als Raspberry Pi 4
+Ethernet-Anschluss +Programmierbare GPIO-Pins
+Garantierte Produktion bis 2026 +4X USB 2 Ports
Contra
—Mikro-USB-Stromanschluss —Aktueller (2022) Preisanstieg

Wählen Sie diesen Computer, wenn Sie keine "Desktop PC Leistung" benötigen, oder eine Budget-Freundliche Alternative benötigen.
Zur Website

Rock Pi 4

Der Rock Pi 4 ist eine beliebte Alternative zu dem Raspberry Pi 4, und verfügt über deutlich mehr Prozessorleistung. Auch wenn Rock Pi kein teil der berühmten Raspberry Pi Foundation ist, so stehen sie dem Marken-Computer in nichts nach. Tatsächlich sind die meisten Rock-Pis ihren Raspberry alternativen um einiges vorraus. Wenn auch leider häufig im Preis.

Pro
+HDMI-Anschluss +USB-C stromanschluss
+Ethernet-Anschluss +Programmierbare GPIO-Pins
+2X USB 3 Ports +2X USB 2 Ports
+Wlan und Bluetooth +Sehr leistungsstarke CPU / GPU
Contra
—Relativ hoher Preis

Wählen Sie diesen Computer, wenn Sie einen Besonders leistungsstarken Computer suchen, mit KI-Technik, oder mit Video-Arbeit auch unterwegs nicht stoppen wollen, oder einfach ein kleines Extra wollen.
Zur Website

Das Touchpad

Als Touchpad kann ich eine klare Empfehlung geben: das offizielle Touchpad der Raspberry Pi Foundation. Dieses sollte mit alle obigen Computer kompatibel sein.

Sollten Sie aus Preisgründen, oder Präferenz ein anderes Touchpad verwenden wollen, so müssen Sie selber auf die Kompatibilität achten, und geg. Teile der Anleitung anpassen.

Im folgenden will ich ein paar Beispiele für mögliche Personalisierungen machen.

Strom Pi3 Hat

Diese praktische Platine lässt sich einfach auf den Pi aufsetzen. Danach verfügt dieser über die Fähigkeit über eine beliebige Stromquelle (6V-61V, max. 3A) versorgt zu werden, und kann durch eine Lithium-Ionen-Batterie (/hier), oder wenn Sie sich wild fühlen über einen regulären 9V Batterieblock versorgt werden.

Lithium

strompi

Hier gibt es nun zwei mögliche Varianten: einmal eine "reguläre" Variante mit 1000mAh, und einmal eine Variante mit mehr Kapazität mit 2000mAh. Beide lassen sich auf die selbe art einfach auf den Srom Pi anstecken lassen, und an den Port hinter dem Lan-Port einstecken lassen. Danach wird sich dieser automatisch aufladen, wenn eine Stromquelle angeschlossen wird, und entladen, wenn der Pi sonst keine Stromquelle hatt.

9V-Batterie

strompi

Nun können Sie eine beliebige Energiequelle zwischen 6 und 61V an die zwei kleinen Klemmen anschließen. Was genau dies sein könnte liegt natürlich Ihnen überlassen. Im meinem Fall wechsele ich gelegentlich zwischen einen kleinem Netzteil mit Kabel, welches eine passende Spannung liefert, und dessen Enden ich abgeschnitten habe, und einer 9V-Batterie.
Der Vorteil von dieser liegt einfach darin, dass ich mein Tablet im Zweifelsfall nicht nachladen muss, sondern einfach die Batteriewechseln kann. Einen Clip können Sie z.B. hier, oder quasi überall anders für einen Fingehut voll Cents bestellen.
